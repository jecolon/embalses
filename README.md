# embalses
Go port of @eocasio 's Python script to fetch Puerto Rico water basin surface level data.

# Got Docker?
If on Linux 64 bit (amd64/x86_64):

    docker run --rm -it jecolon/embalses

# Building
Go 1.5 recomended.
```bash
git clone https://github.com/jecolon/embalses
cd embalses
go build
./embalses
```

# Usage
    ./embalses
